Downloading The Application

Step 1

Navigate to the releases page on GitHub

https://github.com/ksaclan/lion-statics/releases

Step 2

Find the version labeled “Latest Release”

Step 3

Under “Assets”, select “release.apk”

Step 4

After the file has finished downloading, select “open”.

Step 5

When prompted, press “Install”. If the install is blocked, use the troubleshooting guide provided below.

Troubleshooting

Step 1

If installation is blocked, go to your settings

Step 2

Enable 'Allow installation of apps from sources other than the Play Store'

Step 3

Return to your browser and download the Application file again.
